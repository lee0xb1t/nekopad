# NEKOPAD

A bilibili audio player

## Attention

这个软件调用的接口是逆向分析得到的。所以：

* 平台随时可能改动或封堵接口，软件会突然不能用；

* 平台风控系统如果检测到异常调用，可能导致你的账号被限流、禁功能、甚至永久封禁。

这些风险我控制不了，也没法负责。你用这个软件，就代表你接受以上所有可能。

## Features

* Login
* Fetch favorite list
* Fetch video list
* Play audio
